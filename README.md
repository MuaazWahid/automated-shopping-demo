# Automated Shopping Demo
An automated camera shopping script that identifies a good camera deal during the holiday season.

This is an example of software QA utilizing common QA technologies:
- TestNG
- Log4j
- Selenium
- Apache XSSF

### Credit
This script was developed with the help of my partner Gelin Deng

### Files
`PricingData.xlsx`: sample comparable camera prices

`TestCases.xlsx`: sample tests for the script

`emailable-report.html`: report showing each stage of the script successfully executed

`testng.xml`: configuration info for TestNG
